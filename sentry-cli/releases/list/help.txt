sentry-cli-releases-list 
List the most recent releases.

USAGE:
    sentry-cli releases list [OPTIONS]

OPTIONS:
    -D, --delimiter <delimiter>    Delimiter for the --raw flag
    -h, --help                     Prints help information
        --log-level <LOG_LEVEL>    Set the log output verbosity. [possible values: trace, debug,
                                   info, warn, error]
    -R, --raw                      Print raw, delimiter separated list of releases. [defaults to new
                                   line]
    -P, --show-projects            Display the Projects column
    -V, --version                  Prints version information
