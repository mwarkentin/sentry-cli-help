sentry-cli-releases-info 
Print information about a release.

USAGE:
    sentry-cli releases info [OPTIONS] <VERSION>

OPTIONS:
    -h, --help                     Prints help information
        --log-level <LOG_LEVEL>    Set the log output verbosity. [possible values: trace, debug,
                                   info, warn, error]
    -q, --quiet                    Do not print any output.
                                   If this is passed the command can be used to determine if a
                                   release already exists.  The exit status will be 0 if the release
                                   exists or 1 otherwise.
    -V, --version                  Prints version information

ARGS:
    <VERSION>    The version of the release
