sentry-cli-info 
Print information about the Sentry server.

USAGE:
    sentry-cli info [OPTIONS]

OPTIONS:
        --config-status-json       Return the status of the config that sentry-cli loads as JSON
                                   dump. This can be used by external tools to aid the user towards
                                   configuration.
    -h, --help                     Prints help information
        --log-level <LOG_LEVEL>    Set the log output verbosity. [possible values: trace, debug,
                                   info, warn, error]
    -q, --quiet                    Do not output anything, just report a status code for correct
                                   config.
