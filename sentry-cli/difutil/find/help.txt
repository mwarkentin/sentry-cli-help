sentry-cli-difutil-find 
Locate debug information files for given debug identifiers.

USAGE:
    sentry-cli difutil find [OPTIONS] [--] [ID]...

OPTIONS:
    -h, --help                     Prints help information
        --json                     Format outputs as JSON.
        --log-level <LOG_LEVEL>    Set the log output verbosity. [possible values: trace, debug,
                                   info, warn, error]
        --no-cwd                   Do not look for debug symbols in the current working directory.
        --no-well-known            Do not look for debug symbols in well known locations.
    -p, --path <paths>...          Add a path to search recursively for debug info files.
    -t, --type <TYPE>...           Only consider debug information files of the given type.  By
                                   default all types are considered. [possible values: dsym, elf,
                                   pe, pdb, proguard, breakpad, sourcebundle]
    -V, --version                  Prints version information

ARGS:
    <ID>...    The debug identifiers of the files to search for.
