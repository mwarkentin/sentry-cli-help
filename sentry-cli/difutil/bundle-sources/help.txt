sentry-cli-difutil-bundle-sources 
Create a source bundle for a given debug information file

USAGE:
    sentry-cli difutil bundle-sources <paths>...

OPTIONS:
    -h, --help                     Prints help information
        --log-level <LOG_LEVEL>    Set the log output verbosity. [possible values: trace, debug,
                                   info, warn, error]
    -o, --output <PATH>            The path to the output folder.  If not provided the file is
                                   placed next to the input file.
    -V, --version                  Prints version information

ARGS:
    <paths>...    The path to the input debug info files.
